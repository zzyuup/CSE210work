CLASS: Program
Attributes:
- _running : bool
- _choice : string
- _breathing : BreathingActivity
- _reflection : ReflectionActivity
- _listing : ListingActivity
Behaviors:
+ Main(args : string[]) : void
- DisplayMenu() : void
- HandleChoice(choice : string) : void
- Quit() : void
Responsibilities:
- Show the main menu
- Let the user choose an activity
- Run a loop until the user chooses quit
- Call the selected activity
- Restart the menu after each activity ends


CLASS: Activity
Attributes:
- _name : string
- _description : string
- _duration : int
Behaviors:
+ Activity(name : string, description : string)
+ Run() : void
+ DisplayStartingMessage() : void
+ DisplayEndingMessage() : void
+ SetDuration(seconds : int) : void
# PerformActivity() : void
# ShowSpinner(seconds : int) : void
# ShowCountdown(seconds : int) : void
# Pause(seconds : int) : void
Responsibilities:
- Store the common activity name, description, and duration
- Show the common starting message
- Show the common ending message
- Provide shared animation and delay behavior
- Let each derived class do its own activity logic


CLASS: BreathingActivity : Activity
Attributes:
- _breathingPrompts : List<string>
- _randomizer : BreathingPromptRandomizer
- _tracker : BreathingPromptTracker
Behaviors:
+ BreathingActivity()
+ Run() : void
# PerformActivity() : void
- DisplayBreathingPrompt() : void
- ResetPromptRound() : void
Responsibilities:
- Run the breathing activity
- Alternate through breathing prompts for the chosen duration
- Use a randomized display order for the prompt cycle
- Prevent duplicates in one round
- When all prompts have been shown, reshuffle and start a new round
- Return to the menu when finished

PROGRAM FLOW:
1. Program starts.
2. Program creates one BreathingActivity, one ReflectionActivity, and one ListingActivity.
3. Program enters the main menu loop.
4. User chooses an option.
5. If the user chooses Breathing, Program calls BreathingActivity.Run().
6. If the user chooses Reflection, Program calls ReflectionActivity.Run().
7. If the user chooses Listing, Program calls ListingActivity.Run().
8. Each activity:
   - shows the starting message,
   - gets the duration,
   - runs its own activity loop,
   - uses its private randomizer and tracker classes,
   - avoids duplicates in the same round,
   - reshuffles when all prompts have been used,
   - shows the ending message.
9. Control returns to Program.
10. Program shows the menu again.
11. The loop stops only when the user chooses Quit.
